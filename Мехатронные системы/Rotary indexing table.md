## Рабочий цикл

### Функциональные возможности комплекса
1. Принять заготовку
2. Определить наличие/отсутсвие отверстия
3. Проделать отверстие
4. Выдать заготовку

### Рабочие режимы
1. Остановлен
2. Активен в ручном режиме (цикл обработки одной детали)
3. Активен в автоматическом режиме (бесконечный цикл обработки)

### Таблицы сотояний
**Состояния:**
1. Базовое (неактивное)
2. Поворот
3. Рабочий цикл

**Состояния рабочего цикла определяющего станка:**
1. Неактивен
2. Активен на опускание
3. Активен на поднятие
4. Цикл завершен

**Состояния рабочего цикла сверлильного станка:**
1. Неактивен
2. Зажим, включение мотора
3. Опускание сверла
4. Поднятие сверла
5. Выключение мотора, разжим
6. Цикл завершен

**Состояния рабочего цикла выдачи:**
1. Неактивен
2. Выдача
3. Стабилизация
4. Цикл завершен
