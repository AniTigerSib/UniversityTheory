## Системы искусственного интеллекта

Системы искусственного интеллекта - это аппаратно-программные средства, решающие задачу моделирования творческой деятельности человека для решения слабо-формализованных задач.

### Историческая линия

- 1943 год
Американские ученые Уоррен Маккалок и Уолтер Питтс опубликовали научную статью, где рассматривали нейроны с позиции булевой алгебры (вычислительных элементов).
- 1949 год
Канадский психолог Дональд Хэбб сформулировал алгоритм обучения.
- 1959 год
Алгоритм обучения Розенблатта
- 1960 год
Алгоритм обучения Уидроу-Хоффа
- 1986 год
Алгоритм обратного распространения ошибки (BP)

### Математическая модель искусственного нейрона. Перцептрон. Решение логических задач с помощью перцептрона.

Будем считать, что нейрон выполняет функцию нелинейного интегрирования. То есть, нейрон - *нелинейный интегратор*.

#### Математическая модель

![[Pasted image 20260206115345.png]]
Математическая модель содержит $R$ входов.
$P_i$ - входы.
$W_i$ - весовые коэффициенты.
$b$ - фиктивный вход (смещение).
$S$ - взвешенная сумма.
$$S = P_1*W_1+P_2*W_2+...+P_R*W_R+b$$
$y$ - выходной сигнал нейрона.
ФА - функция активации (ниже).
![[Pasted image 20260206115815.png]]
Здесь:
Непрерывной линией - ниполярная функция активации (hardlim)
Пунктиром - биполярная функция активации.

Если у искусственного нейрона функцией активации является ступень, то такой нейрон часто называют **перцептроном**.

Ввиду того, что для ступенчатых функций активации невозможно рассчитать производную в нуле, используют гладкие (сигмоидальные) или колоколообразные функции активации. Или Кусочно-линейные и т.д.

![[Pasted image 20260206120303.png]]

#### Пример
##### 2И
Рассмотрим задачу решения 2И перцептрона.

| $P_1$ | $P_2$ | $Y$ |
| ----- | ----- | --- |
| 0     | 0     | 0   |
| 0     | 1     | 0   |
| 1     | 0     | 1   |
| 1     | 1     | 1   |
*Таблица истинности*

Рассмотрим на графике. а не на таблице:
(пикча)

$$S=W_1P_1+W_2P_2+b=0$$
*Уравнение переключения. Дискриминантная функция.*

$$P_2=f(P_1)$$
*Канонический вид дискриминантной функции.*

Из этого выведем, что для нашей задачи в каноническом виде уравнение переключения будет следующим:

$$P_2=-\frac{W_1}{W_2}P_1-\frac{b}{W_2}$$

Настройка перцептрона заключается в нахождении коэффициентов $W_1, W_2, b$. Частный случай машинного обучения.

#### Алгоритм обучения Хэбба

Сила связи двух активных взаимодействующих нейронов усиливается.
(пикча)

То есть при активном взаимодействии сила связи (вес) с течением времени увеличивается.
$$W(k+1)=W(k)+\Delta{W}$$
$k$ - номер итерации
$\Delta{W}$ - приращение
**Только если ОБА нейрона активны (на выходе 1)!** Соответственно:
$$\Delta{W}=x*y$$

>[!tip] Важно
> Алгоритм обучения Хэбба не гарантирует (процедуры) сходимости!

Данный механизм соответствует парадигме **обучения с учителем**.

### Алгоритм обучения Розенблатта. Алгоритм обучения с подкреплением.

>[!question] Вопрос
> Зачем менять (пересчитывать) весовые коэффициенты, если сеть не ошиблась?

**Основные этапы алгоритма обучения:**
- Инициализация начальных весовых коэффициентов (нули или случайные числа по Розенблатту)
- Определение скорости обучения $\alpha$ в диапазоне $(0;1]$
- На вход сети подаются некоторые данные: $P \to net$
- Вычисляется реакция сети: $net \to Y$
- Реакция сравнивается с эталонным значением:
	- Если они совпадают ($Y_1=T_1$) - весовые коэффициенты остаются без изменений
	- Если они не совпадают ($Y_1\not=T_1$) - весовые коэффициенты пересчитываются: $\Delta w=\alpha pt$
- Повторный прогон обучающей выборки на предмет реакции сети:
	- Если все ответы сети совпадают с эталонными - обучение завершается

## Градиентные алгоритмы обучения
### Алгоритм обучение Уидроу-Хоффа

Данный алгоритм используется для обучения однослойной линейной сети. То есть фенкция активации (TF) - линейная функция.

>[!tip]
>Линейная функция активации позволяет получить любое действительное значение в диапазоне $[0, 1]$ на выходе

В качестве метрики используется средне-квадратичная ошибка (разновидность МНК):
$$E=\frac{1}{2}(y-t)^2\to min$$
$y$ - реакция сети
$t$ - целевое значение

**Основные этапы алгоритма обучения:**
1. Инициализация начальных значений весовых коэффициентов (например, по случайному закону)
2. Необходимо указать шаг (скорость) обучения (для всех градиентных алгоритмов обучения).

Приращение весов:
$$\Delta W=-\alpha \frac{\partial E}{\partial W}$$
$$\frac{\partial E(k)}{\partial w_j(t)} = \frac{\partial E(k)}{\partial y^k}*\frac{\partial y^k}{\partial w_j}=(y^k-d^k)*x^k_j$$
$d$ - целевое значение

![[Pasted image 20260220124321.png]]
#### Проблема градиентных методов

![[Pasted image 20260220123237.png]]
$$\frac{\partial E}{\partial W_i}=0$$
**Данный метод не позволяет определить глобальный экстремум!**

Для нахождения глобального экстремума возсожен к использованию полный перебор, но на практике он не применим (слишком затратный и долгий). На замену ему используется **генетический алгоритм**.

Идея адаптивного шага заключается в том, что:
$$\alpha \sim \frac{1}{k}$$
Чем ближе к экстремуму - тем шаг короче.

Для остановки алгоритма задается максимальный экстремум:
$$E_{max}=10^{-7}$$

Условие выхода из глобального экстремума, не удовлетворяющего нашему $E_{max}$ можно заново инициализировать весовые коэффициенты.
### Алгоритм обратного распространения ошибки (BP)
*BP - Back Propagation*

Данный алгоритм позволяет обучить многослойные нейронные сети прямого распространения (полносвязные структуры).

**Основные этапы:**
1. Инициализация начальных значений весовых коэффициентов.
	1. Рекомендуется использовать случайное распределение в диапазоне $[-0.05;0.05]$ или $[-0.1;0.1]$.
	2. Функция активации сети должна быть дифференцируемой.
		Чаще всего используются сигмоидальные (гладкие) функции. В разных слоях могут использоваться разные функции активации, при этом в рамках одного слоя рекомендуется использовать одинаковые TF.
2. Задаем скорость обучения $\alpha$. Задаем граничное значение ошибки $E_M$.
3. Этап прямого распространения сигнала.
	$y=F(wp+b)$
4. Этап обратного распространения ошибки.
	Для выходного слоя $\gamma_{посл}=y_{реакц}-t_{эталон}$
	Для каждого предыдущего $\gamma=\sum(\gamma_iF'(s)W)$ где $\gamma_i$ - значение предшествующего нейрона
5. Изменение весовых коэффициентов.
	$\Delta W = -\alpha \gamma F'(s)y$
	При начальных значениях вне рамок обозначенных выше - возможен паралич сети, поскольку производная у пороговых (пологих) участков стремится к нулю - соответственно сеть не будет балансироваться.
6. Если ошибка удовлетворяет пороговому значению - останов.

**Недостатки BP:**
1. BP не отвечает на вопрос об архитектуре сети, то есть, *количестве слоев и количестве нейронов в каждом слое*.
2. Возможно перобучение нейронной сети при слишком большом количестве нейронов в вслое/слоев. Характеризуется обобщающей способностью нейронной сети и потерей способности интерполяции данных.
3. Паралич сети связанный с отсутствием уменьшения ошибки (весовые коэффициенты практически не изменяются).

Для решения проблемы переобучения можно ввести эффект прорежевания - разорвать некоторый процент связей между нейронами.
