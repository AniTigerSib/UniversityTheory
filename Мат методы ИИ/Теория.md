## Системы искусственного интеллекта

Системы искусственного интеллекта - это аппаратно-программные средства, решающие задачу моделирования творческой деятельности человека для решения слабо-формализованных задач.

### Историческая линия

- 1943 год
Американские ученые Уоррен Маккалок и Уолтер Питтс опубликовали научную статью, где рассматривали нейроны с позиции булевой алгебры (вычислительных элементов).
- 1949 год
Канадский психолог Дональд Хэбб сформулировал алгоритм обучения.
- 1959 год
Алгоритм обучения Розенблатта
- 1960 год
Алгоритм обучения Уидроу-Хоффа

### Математическая модель искусственного нейрона. Перцептрон. Решение логических задач с помощью перцептрона.

Будем считать, что нейрон выполняет функцию нелинейного интегрирования. То есть, нейрон - *нелинейный интегратор*.

#### Математическая модель

![[Pasted image 20260206115345.png]]
Математическая модель содержит $R$ входов.
$P_i$ - входы.
$W_i$ - весовые коэффициенты.
$b$ - фиктивный вход (смещение).
$S$ - взвешенная сумма.
$$S = P_1*W_1+P_2*W_2+...+P_R*W_R+b$$
$y$ - выходной сигнал нейрона.
ФА - функция активации (ниже).
![[Pasted image 20260206115815.png]]
Здесь:
Непрерывной линией - ниполярная функция активации (hardlim)
Пунктиром - биполярная функция активации.

Если у искусственного нейрона функцией активации является ступень, то такой нейрон часто называют **перцептроном**.

Ввиду того, что для ступенчатых функций активации невозможно рассчитать производную в нуле, используют гладкие (сигмоидальные) или колоколообразные функции активации. Или Кусочно-линейные и т.д.

![[Pasted image 20260206120303.png]]

#### Пример
##### 2И
Рассмотрим задачу решения 2И перцептрона.

| $P_1$ | $P_2$ | $Y$ |
| ----- | ----- | --- |
| 0     | 0     | 0   |
| 0     | 1     | 0   |
| 1     | 0     | 1   |
| 1     | 1     | 1   |
*Таблица истинности*

Рассмотрим на графике. а не на таблице:
(пикча)

$$S=W_1P_1+W_2P_2+b=0$$
*Уравнение переключения. Дискриминантная функция.*

$$P_2=f(P_1)$$
*Канонический вид дискриминантной функции.*

Из этого выведем, что для нашей задачи в каноническом виде уравнение переключения будет следующим:

$$P_2=-\frac{W_1}{W_2}P_1-\frac{b}{W_2}$$

Настройка перцептрона заключается в нахождении коэффициентов $W_1, W_2, b$. Частный случай машинного обучения.

#### Алгоритм обучения Хэбба

Сила связи двух активных взаимодействующих нейронов усиливается.
(пикча)

То есть при активном взаимодействии сила связи (вес) с течением времени увеличивается.
$$W(k+1)=W(k)+\Delta{W}$$
$k$ - номер итерации
$\Delta{W}$ - приращение
**Только если ОБА нейрона активны (на выходе 1)!** Соответственно:
$$\Delta{W}=x*y$$

>[!tip] Важно
> Алгоритм обучения Хэбба не гарантирует (процедуры) сходимости!

Данный механизм соответствует парадигме **обучения с учителем**.

### ### Алгоритм обучения Розенблатта. Алгоритм обучения с подкреплением.

>[!question] Вопрос
> Зачем менять (пересчитывать) весовые коэффициенты, если сеть не ошиблась?

**Основные этапы алгоритма обучения:**
- Инициализация начальных весовых коэффициентов (нули или случайные числа по Розенблатту)
- Определение скорости обучения $\alpha$ в диапазоне $(0;1]$
- На вход сети подаются некоторые данные: $P \to net$
- Вычисляется реакция сети: $net \to Y$
- Реакция сравнивается с эталонным значением:
	- Если они совпадают ($Y_1=T_1$) - весовые коэффициенты остаются без изменений
	- Если они не совпадают ($Y_1\not=T_1$) - весовые коэффициенты пересчитываются: $\Delta w=\alpha pt$
- Повторный прогон обучающей выборки на предмет реакции сети:
	- Если все ответы сети совпадают с эталонными - обучение завершается
