## Основные понятия теории автоматического управления
*Управление* - это процесс выработки и воздействия на объект с целью обеспечения требуемых показателей качества работы системы (быстродействие, точность, надежность и т.д.).

Данная дисциплина рассматривает только технические и технологические объекты.
#### Автоматическое управление
В *автоматическом управлении* процесс функционирования осуществляется без участия человека. В *автоматизированном управлении* присутствует человек-оператор или группа операторов (группа лиц, принимающая решения).

#### Обобщенная структурная схема системы автоматического управления
![[photo_2024-09-04_13-39-05.jpg]]
*УУ* - управляющее устройство (контроллер / промышленный контроллер, регулятор / промышленный регулятор)
*ОУ* - объект управления
*ОС* - обратная связь
*g* - задающее воздействие (уставка) - желаемая величина на выходе системы
*y* - выходная управляемая величина (координата)
*z* - возмущающее воздействие (помеха)
*u* - управляющее воздействие

#### Общий закон управления
$$u = g - y$$
Как правило в системах автоматического управления используется отрицательная обратная связь.
#### Дисциплина ТАУ
**Дисциплина ТАУ** решает две основные задачи:
1) Задача анализа существующей системы, т.е. исследование ее свойств.
2) Задача синтеза (выбрать архитектуру, закон управления, параметр управления)
#### Классификация САУ (систем автоматического управления)
- 
	1. Системы стабилизации ($g = const$: задающее воздействие заранее известно)
	2. Программное управление ($g = var$: задающее воздействие - варьируемая величина, но заранее известная)
	3. Следящие системы ($g = var$: задающее воздействие - варьируемая величина, но заранее неизвестная)
- 
	1. Одномерные системы (одна регулируемая величина)
	2. Многомерные системы (две и боеле регулируемых величин)
- 
	1. Линейные 
	2. Не линейные
- 
	1. Системы с сосредоточенными параметрами (y ~ t)
	2. Системы с распределенными параметрами (y ~ t, координаты)
*Если выходная управляемая величина зависит не только от времени, но и хотя бы от одной координаты, то такая система называется системой с распределенными параметрами.*

### Методы оценки устойчивости линейных систем
#### Корневой метод оценки устойчивости линейных систем
$$W_{3g}(S) = \frac {B(S)}{A_3(S)}$$
$A_3(S) = 0$ - и отсюда ищем корни $S_{1, 2... n}$

*Пример:*

$W_3 = \frac {10}{0,1S^2 + 1}$
$0,1S^2 + 1 = 0$
$S^2 = -1$
$S_{1, 2} = \pm j \omega$

![[Pasted image 20241009134352.png]]
**Система является устойчивой, если все корни характеристического уравнения - "левые" (то есть действительная часть меньше нуля).**
- *Если хотя бы один корень "правый" - система неустойчива;*
- *Если пара мнимых корней - то система на границе устойчивости.*
- *Если __один__ нулевой корень - система на границе устойчивости.*

#### Критерии устойчивости
Под критериями понимают метод, который позволяет оценить устойчивость, **не находя корни** характеристического уравнения.

##### Алгебраический критерий Гурвица
**Необходимое условие устойчивости:**
	Все коэффициенты характеристического уравнения - одного знака.
**Достаточное условие:**
	Все миноры главной диагонали матрицы Гурвица больше нуля.
**Следствие:**
	Для системы не выше 4 порядка доказано, что достаточным условием является $\Delta _{n-1} > 0$

*Методика составления матрицы Гурвица:*
Квадратная матрица размера $n*n$ 
1. В первой строке записываются коэффициенты с нечетным индексом. Недостающие элементы - нулями
	$W_{3g}(S) = \frac {b_0}{a_0S^n + a_1S^{n - 1} + a_2S^{n - 2}}$
2. Вторая строка состоит из коэффициентов с четными индексами, начиная с $a_0$
3. Третья строка состоит из коэффициентов первой строки со сдвигом на 1 элемент.
4. Четвертая строка состоит из коэффициентов второй строки со сдвигом на 1 элемент.
И так далее

*Примеры:*
Пример 1.
$$W_{3g}(S) = \frac {b_0}{a_0S^2 + a_1S + a_2}$$
(пикча)
1. Вывод: система устойчива;
2. Вывод: для системы первого и второго порядков необходимое условие является достаточным.

##### Критерий Михайлова (частотный критерий)
*Система является устойчивой, если [годограф](https://ru.wikipedia.org/wiki/%D0%93%D0%BE%D0%B4%D0%BE%D0%B3%D1%80%D0%B0%D1%84) Михайлова начинается на действительной положительной полуоси и проходит последовательно против часовой стрелки N квадрантов с ростом частоты, где N - порядок системы.*

*Если годограф проходит через начало координат, система будет находиться на границе устойчивости.*

**Методика построения годографа Михайлова:**
$$W_{3g} = \frac {B(S)}{A_3(S)} = \frac {b_0S^m + b_1S^{m - 1} + ... + b_m}{a_0S^n + a_1S^{n - 1} + ... + a_n}$$
$$A_3(S=i\omega) = P(\omega) + iQ(\omega)$$
Если степень нечетная - в наличии будет мнимая составляющая.

**Пример**
$A_3(S) = a_oS^3 + a_1S^2 + a_2S + a_3$
$S=i\omega$
$P(\omega)=-a_1\omega^2+a_3$ : действительная часть
$Q(\omega)=-a_0\omega^3+a_2\omega$ : мнимая часть

Годограф (в комплексной плоскости):
$\omega$ - var

##### Критерий устойчивости Найквиста (Nyquist)
*Замкнутая система устойчива, если для устойчивой разомкнутой системы годограф Найквиста (АФЧХ) не охватывает точку с координатами $(-1;i0)$*

*Система на границе устойчивости, если годограф ТОЧНО ПРОХОДИТ через точку с координатами $(-1;i0)$*

**Методика построения годографа Найквиста**
$$W_p(S)=\frac{k}{TS+1}$$
$$S=i\omega$$
$$W_p(i\omega)=\frac{k}{iT\omega+1}*\frac{1-iT\omega}{1-iT\omega}=\frac{k-ikT\omega}{T^2\omega^2+1}=\frac{k}{T^2\omega^2+1}-i\frac{kT\omega}{T^2\omega^2+1}$$
$H(\omega)e^{i\phi(\omega)}$
$H(\omega)$ - АЧХ (амплитудно-частотная характеристика)
$\phi(\omega)$ - ФЧХ (фазо-частотная характеристика)
$$H(\omega)e^{i\phi(\omega)}=\sqrt{\frac{k^2}{(T^2\omega^2+1)^2}+\frac{k^2T^2\omega^2}{(T^2\omega^2+1)^2}e^{i\phi(\omega)}}=\frac{k}{\sqrt{T^2\omega^2+1}}e^{-i}$$
