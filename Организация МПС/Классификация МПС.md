1. По количеству БИС в микропроцессоре
	1. Однокристальные;
	2. Многокристальные.
2. По назначению
	1. Универсальные - предназначены для решения широкого круга задач;
	2. Специализированные - предназначены для решения узкого круга задач.
3. По виду обрабатываемых сигналов
	1. Цифровые;
	2. Аналоговые.
4. По структурной организации магистрали МПС
	1. Многомагистральные - магистрали адреса и данных реализованы раздельно;
	2. Одномагистральные - магистрали адреса и данных объединены в одну (единую) магистраль.
5. По количеству одновременно выполняемых программ или задач
	1. Однозадачные;
	2. Многозадачные.
6. По принципу организации памяти
	1. МП с классической орагнизацией памяти (структура типа Фон-Неймана). Для хранения программ и данных используется единое адресное пространство. Каждый адрес встречается только один раз;
	2. МП с гарвардской архитектурой. Имеют разделенные адресные пространства памяти для хранения программ и данных. Каждый адрес встречается дважды. Тип как правило применяется в микроконтроллерах.
7. По организации системы команд
	1. CISC (Complete Instruction System Comand) - МП с полным набором системы команд (система команд полного набора/объема);
	2. RISC (Reduced Instruction System Comand) - МП с системой команд сокрущенного набора/объема.
## Схема МПС, обрабатывающей аналоговые системы
```mermaid
flowchart LR

АЦП --> ЦМП
ЦМП --> ЦАП
```
Где:
- **АЦП** - аналогово-цифровой преобразователь
- **ЦАП** - цифрово-аналоговый преобразователь
- **ЦМП** - центральный микропроцессор
